# Project_IoT2_Temp_Light_Control
Dit project bewaakt de temperatuur en het lichtniveau in een ruimte (bijvoorbeeld een kamer) en regelt een ventilator (DC-motor)
en een RGB LED om de omstandigheden te optimaliseren.

### Componenten
* Arduino Nano RP2040
* breadboard
* temperatuursensor VOEG TYPE TOE
* foto transistor
* DC motor (6/9V)
* H-bridge motor driver VOEG TYPE TOE
* RGB LED
* 220 Ohm weerstand voor de LED
* 10k Ohm weerstand voor de phototransistors
